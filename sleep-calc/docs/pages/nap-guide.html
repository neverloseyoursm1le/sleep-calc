Гайд по дневным дремам

Короткие дремы полезны для бодрости: 10–20 минут — для быстрой перезагрузки; 90 минут — чтобы пройти полный цикл.

Когда лучше дремать

Оптимально между 13:00 и 16:00 (у большинства людей снижается бодрость). Избегайте дрем в поздние вечерние часы.

Советы

  • Установите будильник: 20–30 минут для освежения.
  • 90 минут — подойдет, если у вас есть возможность восстановиться и не ложиться поздно вечером.
